# Lidehra MVP — Coach Platform (MGSCC)

MVP de plataforma de coaching executivo baseado na metodologia Marshall Goldsmith Stakeholder Centered Coaching.

## Instalação local
```bash
npm install
npm run dev
```

## Deploy no Vercel
1. Faça push para o GitHub
2. Importe o repositório em vercel.com
3. Deploy automático
